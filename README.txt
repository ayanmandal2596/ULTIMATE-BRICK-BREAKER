Instructions:  
Download the tar.gz file from www.moodle.iitb.ac.in and extract the contents - report, source, presentation.
open the source folder to view the code. Open this code using Code::Blocks and click on the option 'Complie and Run'.
Our project is a software project, so you don’t any hardware component other than a computer system.
To play this game you should have a system having Code::Blocks installed in it along with SimpleCPP graphics library.
1) 	 Download Code::Blocks from 
 http://www.codeblocks.org/downloads/26#windows

2)	 Code::Blocks and SimpleCPP installation manual available at
http://www.it.iitb.ac.in/frg/wiki/images/e/e8/CodeBlockManual.pdf 

3)	 Video link for setting up things to be able to run this software 
https://www.youtube.com/watch?v=EI1Yow1hyx8
